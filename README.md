TheOdysseyGame

Repository της πτυχιακής εργασίας «Σχεδιασμός και ανάπτυξη βιντεοπαιχνιδιού βασισμένου στην Οδύσσεια του Ομήρου: προσαρμογή ενός κλασικού έπους σε διαδραστικό ψηφιακό περιβάλλον».

Περιεχόμενα του Repository:

OdysseyDemo(.exe): Περιέχει το εκτελέσιμο αρχείο του παιχνιδιού για Windows. Για να εκτελέσετε το demo, κατεβάστε όλο τον φάκελο και εκκινήστε το αρχείο .exe.

OdysseyDemo (unityFiles): Περιέχει τα πλήρη αρχεία του Unity project, συμπεριλαμβανομένων των scenes, scripts, assets και λοιπών αρχείων που χρησιμοποιήθηκαν για την ανάπτυξη του πρωτοτύπου.

Απαιτήσεις
Windows 10 ή νεότερο (για εκτέλεση του demo)
Unity (για άνοιγμα και επεξεργασία του project)

Σκοπός
Το έργο αποτελεί λειτουργικό πρωτότυπο (demo) που αναπτύχθηκε στο πλαίσιο πτυχιακής εργασίας στο Τμήμα Τεχνών Ήχου και Εικόνας του Ιονίου Πανεπιστημίου και διερευνά τη μεταφορά της Οδύσσειας του Ομήρου σε μορφή διαδραστικού τρισδιάστατου βιντεοπαιχνιδιού.
